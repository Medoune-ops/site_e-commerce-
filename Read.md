Maket bi est une application e-commerce "Full-Stack" moderne et performante, bâtie sur une architecture entièrement découplée pour garantir une séparation stricte entre l'interface utilisateur et la logique métier.

Le Back-End est une API REST robuste développée en Node.js, Express et TypeScript, utilisant l'ORM TypeORM pour interagir de manière fluide avec une base de données MongoDB. Cette API intègre une architecture professionnelle sécurisée comprenant une gestion de l'authentification (/auth), un CRUD complet pour les utilisateurs (/users) et un CRUD dédié aux produits (/products) pour stocker les prix, caractéristiques et visuels des articles. Elle est entièrement documentée via Swagger, sécurisée contre les blocages de requêtes inter-origines grâce au middleware CORS, et surveillée par un système global de gestion des erreurs et de logs (Winston).

Le Front-End, quant à lui, est une vitrine commerciale épurée et entièrement responsive, codée nativement en HTML5, CSS3 et JavaScript (ES6+). L'interface respecte scrupuleusement les standards UX/UI actuels à travers une structure soignée : une section Hero percutante mettant en valeur un produit phare découpé dans un cercle parfait, une barre d'avantages clients (Features Bar), un système de navigation par catégories graphiques, ainsi qu'une bannière promotionnelle avec compte à rebours et une section de témoignages clients.

Le cœur dynamique du projet repose sur la connexion de ces deux univers : via l'API Fetch, le script du site communique en temps réel avec le serveur Node.js sur le port 3000 pour récupérer les données stockées dans MongoDB. Le JavaScript injecte ensuite dynamiquement les produits dans une grille moderne gérée en CSS Grid, transformant ainsi une simple maquette statique en une plateforme de commerce fonctionnelle et évolutive.
